# Sitecrawlr

## Description
Crawl sitemaps and filter the url-list based on query selectors or search terms

## Usage

To install sitecrawlr from npm, run:

```
$ npm install -g sitecrawlr
```

```node ./bin/sitecrawlr --help```

## Testing

```
mocha test
```
